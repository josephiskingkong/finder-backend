# Finder Backend — ИИ-наставник для начинающих предпринимателей

Бэкенд приложения для поэтапного сопровождения предпринимательских проектов
на основе методологий Customer Development и Lean Startup.

## Модели данных

### User
Пользователь с JWT-аутентификацией. При регистрации автоматически создаётся
`EntrepreneurProfile` с типом `UNKNOWN`.

### EntrepreneurProfile
Профиль предпринимателя с типом (бизнес-ориентированный / проблемно-ориентированный).
Тип определяется в ходе онбординг-диалога с ИИ.

### Business
Бизнес-проект пользователя. Содержит все данные:
- Описание идеи, проблема, ЦА
- Данные о конкурентах, монетизации
- Юридические данные (ИНН, ОГРН, форма, система налогообложения, ОКВЭД)
- Результаты интервью, описание MVP

### Roadmap + RoadmapStep
Персонализированный роадмап в стиле Duolingo. Шаги:
- `LOCKED` → `AVAILABLE` → `IN_PROGRESS` → `COMPLETED` / `FAILED`
- Каждый шаг принадлежит фазе (проблематизация, рынок, интервью и т.д.)
- Пользователь отчитывается — ИИ анализирует и открывает следующий шаг

### Conversation + Message
Чат с ИИ. Каждая беседа привязана к бизнес-проекту. ИИ получает полный контекст
проекта и роадмапа при каждом запросе.

## API Эндпоинты

### Аутентификация `/api/auth`
| Метод | Путь       | Описание                |
|-------|-----------|-------------------------|
| POST  | /register | Регистрация             |
| POST  | /login    | Вход                    |
| POST  | /refresh  | Обновление токенов      |
| POST  | /logout   | Выход                   |
| GET   | /me       | Данные текущего юзера   |

### Бизнес-проекты `/api/businesses`
| Метод  | Путь                          | Описание                        |
|--------|------------------------------|---------------------------------|
| POST   | /                            | Создать бизнес-проект           |
| GET    | /                            | Все проекты юзера               |
| GET    | /:id                         | Детали проекта                  |
| PATCH  | /:id                         | Обновить проект                 |
| DELETE | /:id                         | Удалить проект                  |
| PATCH  | /profile/entrepreneur-type   | Установить тип предпринимателя  |

### Чат `/api/chat`
| Метод  | Путь                                      | Описание                |
|--------|--------------------------------------------|-------------------------|
| POST   | /message                                   | Отправить сообщение ИИ  |
| GET    | /business/:businessId/conversations        | Список бесед            |
| GET    | /conversations/:conversationId/messages    | Сообщения беседы        |
| DELETE | /conversations/:conversationId             | Удалить беседу          |

**Стриминг:** При `stream: true` ответ приходит через SSE (Server-Sent Events).

### Роадмап `/api/roadmap`
| Метод | Путь                      | Описание                     |
|-------|---------------------------|------------------------------|
| POST  | /generate                 | Сгенерировать роадмап (ИИ)   |
| GET   | /business/:businessId     | Получить роадмап             |
| PATCH | /steps/:stepId/start      | Начать работу над шагом      |
| POST  | /steps/:stepId/report     | Отчитаться по шагу           |

### ФНС `/api/fns`
| Метод | Путь               | Описание                  |
|-------|---------------------|---------------------------|
| GET   | /search/inn/:inn    | Поиск по ИНН              |
| GET   | /search/ogrn/:ogrn  | Поиск по ОГРН             |
| GET   | /search/name?name=  | Поиск по названию         |
| GET   | /company/:inn       | Подробности о компании    |
| GET   | /okved/popular      | Популярные коды ОКВЭД     |

### Файлы `/api/files`
| Метод  | Путь                      | Описание                |
|--------|---------------------------|-------------------------|
| POST   | /upload                   | Загрузить файл          |
| GET    | /business/:businessId     | Файлы бизнес-проекта    |
| DELETE | /:id                      | Удалить файл            |

## Пользовательский флоу

```
1. Регистрация/Вход
       │
2. Онбординг (чат с ИИ)
   ├── "Хочу придумать идею" → Генерация идеи
   └── "У меня есть идея" → Описание бизнеса
       │
3. ИИ определяет тип предпринимателя
   ├── Бизнес-ориентированный → Больше шагов по проблематизации
   └── Проблемно-ориентированный → Больше шагов по бизнес-структуре
       │
4. Генерация персонального роадмапа (10-20 шагов)
       │
5. Работа по шагам (как Duolingo)
   ├── Проблематизация
   ├── Изучение продукта
   ├── Анализ рынка (+ данные ФНС)
   ├── Модель монетизации
   ├── Пользовательские интервью (кастдев)
   ├── MVP
   ├── Повторный кастдев
   ├── Регистрация ИП/ООО
   └── Бухгалтерия и налоги
       │
6. На каждом шаге:
   ├── Чат с ИИ для консультаций
   ├── Загрузка файлов/документов
   └── Отчёт → ИИ анализирует → Корректирует план
```

## Принципы работы ИИ

1. **Честность** — не расхваливает идеи без оснований
2. **Методология** — Customer Development + Lean Startup
3. **Российский контекст** — законы, налоги, ФНС, ОКВЭД
4. **Два типа предпринимателей** — разные подходы к каждому
5. **Контекстуальность** — ИИ всегда знает о проекте и текущем шаге

## Запуск

```bash
# 1. Скопировать конфиг
cp .env.example .env
# Заполнить .env (DATABASE_URL, OPENAI_API_KEY, и т.д.)

# 2. Установить зависимости
npm install

# 3. Применить миграции
npm run db:migrate

# 4. Запустить в dev-режиме
npm run dev

# 5. Или собрать для продакшена
npm run build
npm start
```

## Технологии

- **Runtime:** Node.js + TypeScript
- **Framework:** Express.js
- **ORM:** Prisma
- **БД:** PostgreSQL
- **ИИ:** OpenAI GPT-4o API
- **Аутентификация:** JWT (access + refresh tokens)
- **Загрузка файлов:** Multer
- **Валидация:** Zod
- **Безопасность:** Helmet, CORS, Rate Limiting
